# Night Bull

Simple Python application to experiment with venv, dependencies, pytest,
flake8 and FastAPI.

## Getting Started

```console
$ docker build -t nightbull:0.1
```

## Usage

```console
$ docker run --rm -it -p 8080:8080 nightbull:0.1
```
